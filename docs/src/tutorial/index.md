## How to open the attribute table and perform a query

In this tutorial, we will walk you through all the steps to open the attribute table and perform a query in IGRAC's GGMN (Global Groundwater Monitoring Network) & GGIS (Global Groundwater Information System).

The attribute table is full of potentially interesting information. It allows you to search layers based on year, author, country, etc. This tutorial will help you to take most out of the groundwater data.

## How to download a layer

In this tutorial, we will walk you through all the steps to download a layer in IGRAC's GGMN (Global Groundwater Monitoring Network) & GGIS (Global Groundwater Information System).

We will explain how to download either polygon or raster layers. In the first example, we show how to quickly download a shapefile. After that, you will see how to export data as GeoJSON, CSV, TIFF, etc.
